# Xizmat narxlari

## Asosiy qoidalar

- Mahalliy mijozlar uchun narxlar KAMROQ bo'lishi mumkin, chunki to'lov oson va raqobat past.
- Xalqaro mijozlar (Upwork/Fiverr) uchun narxlar YUQORI, lekin platforma komissiyasi (20%) va to'lov konversiyasi bor.
- Har doim 50% oldindan to'lov yoki to'liq avans talab qiling.

## Tez narx taklifi formulasi

```
Narx = (soatlik daromad x soat) x 1.3
```

Masalan: agar soatlik daromadingiz $10 bo'lsa, vazifa 2 soat olsa:
$10 x 2 = $20 x 1.3 = $26 (taklif qilishingiz mumkin bo'lgan narx)

1.3 koeffitsienti — pul o'tkazish, platforma komissiyasi va neko'r payt uchun zaxira.
