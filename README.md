# AI-Freelance Daromad Boshlang'ich To'plami

Bu to'plam O'zbekistondan chiqib, dunyo bo'ylab mijozlarga xizmat ko'rsatish uchun tayyorlangan. AI tezlik vositasi sifatida ishlatiladi, lekin sifat uchun inson nazorati talab etiladi.

**Diqqat**: Bir kunda $100 va'da qilmayman. Real maqsad: 1-3 kun ichida birinchi kichik order, 1 haftada 3-5 order, 1 oyda $200-$800.

## Ichidagi fayllar

- `fiverr-gigs.md` — tayyor gig sarlavhalari va tavsiflari
- `upwork-proposals.md` — mijozga yuboriladigan taklif namunalari
- `ai-prompts.md` — xizmatlarni tezlashtiruvchi promptlar
- `service-price-list.md` — narxlar (USD va UZS da)
- `local-client-outreach.md` — Telegram/Instagram mijozlariga xabarlar
- `action-checklist.md` — 7 kunlik ish rejasi
- `portfolio.html` — mijozlarga ko'rsatiladigan oddiy portfolio sahifasi
- `telegram-bot-starter.py` — sotilishi mumkin bo'lgan oddiy Telegram bot namunasi
- `invoice-template.md` — mijozlarga yuboriladigan hisob-faktura namunasi
- `requirements.txt` — bot ishga tushirish uchun paketlar
- `.env.example` — muhit o'zgaruvchilari namunasi
- `samples/` — mijozlarga darhol ko'rsatilishi mumkin bo'lgan tayyor namunalar (restoran SMM, blog, tarjima)
