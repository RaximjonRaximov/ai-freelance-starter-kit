# AI Promptlari — Freelance xizmatlar uchun

## 1. Blog post yozish

```
You are an experienced SEO blog writer. Write a [word count]-word blog post on "[topic]" for an audience of [target audience].

Requirements:
- Use a friendly but professional tone
- Include these keywords naturally: [keyword 1], [keyword 2]
- Structure with H2 and H3 headings
- Start with a strong hook
- End with a clear call to action
- Use short paragraphs and bullet points
```

## 2. SMM postlarini yaratish

```
You are a social media manager for [business type]. Create 7 Instagram/Telegram posts for [brand name].

For each post provide:
- Hook (first sentence)
- Main caption (2-5 sentences)
- 5 relevant hashtags
- Call to action
- Suggested image description

Brand voice: [professional/friendly/luxury/funny]
Niche: [niche]
```

## 3. Tarjima va tahrirlash

```
You are a professional translator. Translate the following text from [source language] to [target language].

Then:
- Improve natural flow for native readers
- Keep the original formatting
- Suggest 5 key terms for a glossary
- Note any cultural references that may need adaptation

Text:
[paste text here]
```

## 4. Email marketing

```
Write a 3-email sequence for [business type] to [goal: welcome / abandoned cart / promotion].

Audience: [describe audience]
Tone: [friendly / professional / urgent]
Each email should be 100-150 words with one clear CTA.
```

## 5. Mahalliy mijozga taklif xati (O'zbekcha)

```
Write a short, friendly message in Uzbek for a local [business type, e.g. restaurant, clothing store] offering social media content services. Include:
- A polite greeting
- One clear benefit
- A free sample offer
- A question to continue the conversation

Keep it under 120 words.
```

## 6. Freelance profil bio

```
Write a compelling Fiverr/Upwork profile bio for a freelancer offering [services].

Include:
- What problems they solve
- Why clients should trust them
- What industries they serve
- A friendly call to action

Keep it under 150 words.
```

## 7. Ovoz dublyaj matni

```
You are a voiceover script editor. Review the following script for clarity and natural spoken flow.

- Break long sentences into shorter ones
- Mark pronunciation notes if needed
- Suggest tone: [energetic/calm/professional]
- Final length target: [X] words

Script:
[paste script]
```

## 8. Narx taklifini hisoblash

```
I need to price a freelance project. Details:
- Service: [content writing / translation / SMM / bot]
- Scope: [word count / number of posts / features]
- My target hourly rate: $[X]
- Estimated time: [Y] hours

Calculate a fair fixed-price quote. Also provide a 3-tier package breakdown (Basic, Standard, Premium).
```
